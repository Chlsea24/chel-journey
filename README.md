# Chel Journey
A modern, responsive portfolio website built with **Next.js (App Router)**, **TypeScript**, and **Framer Motion**.

This portfolio showcases my work in **Informatic: Computer Vision, Machine Learning, and Web Development**, combining clean UI design with smooth user experience.

## Tech Stack
- Next.js (App Router)
- TypeScript
- CSS Modules
- Framer Motion
- Vercel (Deployment)

## Features
- Responsive design (mobile-first)
- Smooth scroll animations
- Animated sections with Framer Motion
- Modular and scalable structure
- Clean dark theme UI
